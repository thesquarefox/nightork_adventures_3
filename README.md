# Nightork Adventures 3
This is the source code of Nightork Adventures 3, together with the level editor.

## Code

The source code is written in C++ and provided as Visual Studio 2022 projects.

## Dependencies

The following libraries are used:
- Irrlicht 1.8.5
- Irrklang 1.6.0
- Steamworks SDK 1.64
